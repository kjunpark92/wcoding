React and React Native

This course will take students knowledge of Javascript and take it to an absolute level by teaching them two of the most popular libraries: React and React Native for building websites and cross-platform mobile applications.

Best way to learn is by building, publishing, hosting projects. Below are the shortlist of projects that students might get to build by taking this course.

Projects

Vanilla JS and React

  • Github profile finder with Github API
  • Jobs website with third part APIs
  • News website in English, Russian and Korean
  • Library in the browser
  • Small todos, pokemon list, random quote and jokes websites

React Native

  • Reviews app
  • Note Taking app
  • Online market app
  • Turn the news website into a cross-platform app
  • Capstone project: one of those â†’ voice recognition or messaging or dating app

Class schedule and content

I will be creating this class schedule to teach React and React Native to people who have intermediate familiarity with JavaScript. No knowledge of React is required. This schedule is for a span of 10-12 weeks with three times a week, 2-3 hours per class


Week 1

Topic:

Asynchronous JS, ES6, DOM Manipulation


Goal

  • Get a better idea of everyone's familiarity with JavaScript
  • Explain parts of JS where they have difficulty
  • Build this project with Github API to search user profiles
  • Home assignment: Book Library in the browser with JS

SUMMARY: This week content shall improve students knowledge of JS since React and React Native is JS after all. They shall also know why React was created since building a project with Vanilla Javascript is complex to update and maintain.

Week 2

Topic:

React Syntax, React Components, State


Goal

  • Introduce React, its syntax, why it was created
  • Explain types of components, structure and use cases
  • State and their flow in React.
  • DOM events in React
  • Recreate our GITHUB FINDER project with React

SUMMARY: This week, students should get a first idea about React and recreate the project that was built with Vanilla Javascript. It teaches them how React makes it easy to build projects.

Week 3

Topic:

Func/Class Components, Forms, SPA, Props, Style

Lifecycle methods, React Navigation, Programming Redirects

Goal

  • More in depth React. Build more simple projects
  • Explain updated version and changes in React
  • Why functional components are overtaking class components
  • Styled components
  • Projects: Pokemon list website in React

SUMMARY: This week students have to understand React more in depth and be somehow ready to build and change React projects.

Week 4

Topic:

React HOC, Data Fetching, Axios, Redux


Goal

  • More React and state management
  • Redux, Context, Hooks
  • Projects: News website and Note Taking website in React

SUMMARY: This will probably be hard and important for students. It might take more than a week. Since almost anything they will learn here will be used in React Native.

Week 5

Topic:

React Review, More Redux and Context, Bootstrap


Goal

  • Review what we learned in React so far. Make sure students are clear of React and its state management with React and Context
  • Explain React Bootstrap, React design libraries
  • More into external packages in React
  • Polish our news website and add language compatibility

SUMMARY: Review and strengthen React knowledge of students

Week 6

Topic:

React Native, Expo, Mobile development


Goal

  • Introduce React Native and Expo
  • Native components
  • Native API
  • State and Forms
  • Build TO DO app in React Native

SUMMARY: Introduce React Native and Expo. Its pros and cons. Build a couple short projects in React

Week 7

Topic:

React Native styles, state management


Goal

  • More into state management, themes, data fetching
  • Start an online market app in React Native

SUMMARY: More in depth React Native and the rest of the course from here on will be on Project building

Week 8

Topic:

React Navigation, Formik and Yup


Goal

  • More in depth React navigation
  • Taking forms with Formik and validation with Yup
  • Wrap up
  • Continue our online market app in React Native

SUMMARY: Wrapping up React Native and detailed explanations of React navigation, Formik and yup

Week 9

Topic:

Finalized the market app, start news app, testing


Goal

  • Finalize and how to publish our app to Google Play Store
  • Start news app with a third party API

SUMMARY: Build more projects

Week 10

Topic:

Experience with different APIs and packages


Goal

  • Polish and more work on news app
  • Take a group idea and make a final app in discussion with students
  • Capstone Project: e.g Voice recognition app, messaging app

SUMMARY: This week, we will probably work on project building, testing, maintenance and give students a capstone project to work on by themselves

Week 11

Topic:

Testing, Review capstone projects, Final exam


Goal

  • News in React Native, Famous websites and libraries
  • Review capstone projects, work more on them
  • Final exam

SUMMARY: Final chapter in the course
